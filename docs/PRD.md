# TextSnag 产品需求文档

> 版本: 1.0.0 | 日期: 2026-07-16 | 状态: MVP 已实现

## 概述

TextSnag 是一款 Chrome 浏览器扩展，用户点击工具栏图标后拖拽框选页面区域，即可提取该区域内 DOM 文字内容，并在浮窗中查看、复制结果。

## 核心功能

### 1. 激活方式

| 决策 | 结论 |
|------|------|
| 入口 | 仅工具栏图标点击（后续可加快捷键） |
| 激活后行为 | 进入框选模式，显示半透明遮罩 |

### 2. 选择区域

| 决策 | 结论 |
|------|------|
| 选择方式 | 拖拽框选 |
| 附加模式 | 无（MVP 不支持元素悬停选择） |
| 最小选区 | 10px × 10px（防止误触） |

### 3. 视觉反馈

| 决策 | 结论 |
|------|------|
| 遮罩方案 | 半透明黑色遮罩 + 镂空选区（4-div 拼接） |
| 选区边框 | 蓝色虚线 |
| 尺寸提示 | 选区右下角显示宽×高（px） |
| 光标 | 十字准心（crosshair） |

### 4. 文字提取

| 决策 | 结论 |
|------|------|
| 提取方式 | DOM 文本提取（非 OCR） |
| 格式化策略 | 智能格式化 |
| 块级元素 | 换行分隔 |
| 行内元素 | 直接拼接 |
| 列表项 | 前缀 "• " 加换行 |
| 图片 | 提取 `alt` 属性，无 alt 时显示 `[图片]` |
| 链接 | 仅提取文字，不附带 URL |
| 代码块 | 保留原始缩进和空格 |
| 表格 | 按 DOM 顺序拼接文字 |
| 隐藏元素 | 跳过（`display:none`、`visibility:hidden`、`opacity:0`） |
| 零尺寸元素 | 跳过 |

### 5. 结果展示

| 决策 | 结论 |
|------|------|
| 展示方式 | 浮动弹窗（不自动复制） |
| 弹窗位置 | 选区右下方，视口不足时自动避让 |
| 关闭方式 | × 按钮 + 点击弹窗外任意位置 + ESC |
| 弹窗内容 | 可滚动文字区域 + 字符计数 + 复制按钮 |
| 复制后反馈 | 按钮变为"已复制 ✓"，2 秒后恢复 |
| 容错 | `navigator.clipboard.writeText` 失败时回退到 `execCommand("copy")` |

### 6. 交互行为

| 决策 | 结论 |
|------|------|
| 重新选区 | 不支持。需 ESC 取消后重新点击图标 |
| 编辑文字 | 不支持。浮窗内文字只读 |
| 国际化 | MVP 仅中文界面 |
| 选区时滚动 | 禁止页面滚动 + 禁止文字选中 |
| ESC 键 | 框选模式下取消选区；结果模式下关闭弹窗 |

### 7. 深色模式

| 决策 | 结论 |
|------|------|
| 支持方式 | CSS 自定义属性 + `prefers-color-scheme` 媒体查询 |
| 适配范围 | 遮罩、选区边框、弹窗、按钮等全部 UI 元素 |

### 8. 技术选型

| 决策 | 结论 |
|------|------|
| 框架 | Plasmo（React + TypeScript） |
| 目标平台 | Chrome Manifest V3 |
| 构建工具 | Plasmo 内置（基于 Parcel） |
| 图标 | 自绘 SVG → PNG（16/32/48/64/128px） |
| 第三方依赖 | 无（仅 React + ReactDOM） |

### 9. 权限

| 权限 | 原因 |
|------|------|
| `activeTab` | 仅在用户点击图标时获得当前标签页访问权 |
| `scripting` | Content Script 注入所需 |
| `clipboardWrite` | 写入剪贴板 |
| `storage` | 预留未来设置持久化 |
| `host_permissions: <all_urls>` | 所有页面均可使用 |

### 10. 已知限制

- 无法提取 `<canvas>`、SVG 文本节点、图片内文字（无 OCR）
- 无法提取跨域 iframe 内容
- 不注入 `chrome://` 及 `chrome-extension://` 页面
- 选区时 CSS 动画可能导致遮罩定位抖动
