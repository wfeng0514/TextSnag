📝 段落文字

TextSnag 是一个轻量级的浏览器扩展，让你能够快速提取网页上任意区域的文字内容。只需点击图标、拖拽选区、复制文字，三步完成。支持智能格式化——段落保留换行、列表自动加项目符号、代码块保留格式。

这对于需要从网页上摘录信息的场景特别有用，比如收集研究资料、保存设计规范、摘录文档片段等等。不再需要手动选中、复制、粘贴——TextSnag 让这一切变得丝滑流畅。

📋 列表内容

  • Chrome 扩展开发最佳实践
  • 支持 Manifest V3 规范
  • 使用 Plasmo 框架构建
  • React + TypeScript 技术栈
  • 支持深色模式自适应
